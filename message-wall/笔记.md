#### 1. vite 初始化项目

```
1 创建项目：npm create vite@latest message-wall -- --template vue
2 安装依赖：npm install
3 启动项目：npm run dev
```

#### 2. 安装插件

```
1 安装 vue-router，并配置路由
2 安装 vuex,并配置仓库
3 安装 axios,并进行二次封装
4 安装 le's's
```

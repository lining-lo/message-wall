#### 1. 项目初始化

```
1 初始化项目：npm install express --save
2 项目目录
	|-- serve/
		|-- jsconfig.json
		|-- index.js
		|-- 笔记.md
		|-- config/
		|	|-- default.js
		|-- controller/
		|-- lib/
		|	|-- db.js
		|-- routes/   
3 安装插件
	下载 mysql 插件（连接数据库）：npm install mysql --save
	下载 multer 插件（上传图片）：npm install multer --save
	下载 ejs 插件（解析 html）：npm install ejs --save
```
